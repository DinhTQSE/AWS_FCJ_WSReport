## Tổng quan dự án

Nhật ký công việc này ghi lại **hành trình học tập 12 tuần** (tháng 9 - tháng 11 năm 2025) tập trung vào việc thành thạo **AWS Cloud Computing** thông qua phát triển thực tế một Hệ thống Quản lý Thiết bị IoT Thông minh. Dự án đóng vai trò là phương tiện thực hành để học và áp dụng các dịch vụ AWS cốt lõi, nguyên lý kiến trúc đám mây và các phương pháp DevOps hiện đại.

### Mục tiêu học tập

Mục tiêu chính là đạt được sự hiểu biết toàn diện về **các kiến thức nền tảng và dịch vụ cốt lõi của AWS Cloud Computing** thông qua thực hành:

**Các Dịch vụ AWS Cốt lõi:**
- **Compute**: EC2, Lambda (serverless functions)
- **Storage**: S3 (object storage), EBS, CloudFront (CDN)
- **Database**: DynamoDB (NoSQL), các khái niệm RDS
- **Networking**: VPC, Security Groups, Route 53
- **Security & Identity**: IAM (users, roles, policies), WAF, Shield
- **Monitoring & Management**: CloudWatch (metrics, logs, alarms)
- **IoT & Integration**: IoT Core, API Gateway, EventBridge
- **Developer Tools**: CodePipeline, CodeBuild, Amplify

**Nguyên tắc Kiến trúc Đám mây:**
- **Infrastructure as Code** (Terraform) cho tự động hóa provisioning
- **Các mẫu kiến trúc serverless** và best practices
- **Thiết kế ưu tiên bảo mật** với quyền truy cập tối thiểu
- **Chiến lược tối ưu hóa chi phí** cho tài nguyên đám mây
- **Các mẫu thiết kế high availability & scalability**
- **Triển khai AWS Well-Architected Framework**

### Tiến độ phát triển

Dự án tuân theo phương pháp agile với các sprint hàng tuần, tiến triển từ kiến thức cơ bản về AWS đến triển khai sẵn sàng cho production:

**Tuần 1-7:** [Giai đoạn Nền tảng & Lên kế hoạch](1.1-week1/) - Kiến thức cơ bản AWS, thiết kế kiến trúc, lên kế hoạch dự án

**Tuần 8:** [Phần cứng & Tích hợp IoT](1.8-week8/) - Phát triển pipeline cốt lõi, tích hợp cảm biến, thiết lập AWS IoT Core

**Tuần 9:** [Xác thực & Phát triển API](1.9-week9/) - Hệ thống xác thực lai, Lambda functions, cấu hình API Gateway

**Tuần 10:** [Phát triển & Triển khai Frontend](1.10-week10/) - Giao diện web, triển khai S3/CloudFront, kiểm thử end-to-end

**Tuần 11:** [Infrastructure as Code & CI/CD](1.11-week11/) - Triển khai Terraform, tự động hóa pipeline, củng hóa bảo mật (WAF, giám sát)

**Tuần 12:** [Kiểm thử & Tài liệu](1.12-week12/) - Tối ưu hóa hệ thống, kiểm thử cuối cùng, hoàn thiện tài liệu và bàn giao dự án

### Kết quả học tập đạt được

✅ **Kinh nghiệm thực hành** với hơn 10 dịch vụ AWS cốt lõi
✅ **Làm chủ kiến trúc serverless** - Lambda, API Gateway, DynamoDB
✅ **Triển khai IoT data pipeline** với AWS IoT Core và rules engine
✅ **Triển khai bảo mật đám mây** - IAM policies, WAF rules, mã hóa
✅ **Thành thạo Infrastructure as Code** - Terraform cho quản lý tài nguyên AWS
✅ **Kỹ năng tự động hóa CI/CD** - CodePipeline, quy trình triển khai Amplify
✅ **Giám sát & quan sát đám mây** - CloudWatch dashboards, alarms và logs
✅ **Kiến trúc tiết kiệm chi phí** - Serverless computing và dịch vụ pay-per-use

### Công nghệ sử dụng

**Dịch vụ đám mây:** AWS IoT Core, Lambda, API Gateway, DynamoDB, S3, CloudFront, CloudWatch, WAF, Shield

**Hạ tầng:** Terraform, AWS CodePipeline, AWS Amplify

**Phần cứng:** ESP32, Cảm biến MQ-3, Cảm biến MAX30102, Module vân tay AS608

**Frontend:** HTML5, CSS3, JavaScript (Vanilla)

**Phát triển:** Git, VS Code, AWS CLI

Nhật ký công việc này cung cấp tài liệu chi tiết theo tuần về **hành trình học tập AWS**, bao gồm các bài tập thực hành, thách thức gặp phải, các dịch vụ AWS được khám phá và kỹ năng thực tế đạt được trong suốt chương trình cloud computing 12 tuần.
