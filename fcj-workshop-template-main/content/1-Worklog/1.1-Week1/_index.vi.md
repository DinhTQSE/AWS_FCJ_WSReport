### Kết quả đạt được Tuần 1:

* Làm quen với các thành viên FCJ và hiểu quy trình làm việc của nhóm.
* Hoàn thành các module lý thuyết:

  * Module 01-01: Hiểu khái niệm cơ bản về Điện toán đám mây.
  * Module 01-02: Tìm hiểu điều gì làm cho AWS độc đáo so với các nhà cung cấp cloud khác.
  * Module 01-03: Hiểu cách bắt đầu hành trình AWS Cloud.
* Khám phá AWS Free Tier, AWS Console, và AWS CLI.
* Tạo thành công tài khoản AWS Free Tier và cấu hình cài đặt thanh toán.
* Thiết lập MFA (Multi-Factor Authentication) cho root account.
* Tạo Admin IAM group và Admin IAM user cho công việc hàng ngày.
* Thực hành phân biệt giữa Root User và IAM User, và học các thực hành tốt nhất cho bảo mật tài khoản.
* Hoàn thành Worklog Tuần 1 và bắt đầu đưa ra ý tưởng cho đề xuất dự án.
